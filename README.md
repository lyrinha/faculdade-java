# atividades-faculdade-ads

Esse projeto foi feito como parte das atividades da faculdade (Análise e Desenvolvimento de Sistemas - IBMR).

A ideia foi praticar lógica de programação usando Java e entender melhor como resolver problemas com código.

## O que o programa faz
- Executa as funções propostas no exercício
- Recebe dados do usuário
- Processa as informações e mostra o resultado

## O que eu pratiquei
- Estruturas condicionais (if/else)
- Estruturas de repetição (for/while)
- Entrada e saída de dados
- Lógica de programação

## Como rodar
É só abrir o código em alguma IDE (IntelliJ, Eclipse ou VS Code) e executar a classe principal.

## Observação
Esse é um projeto simples, feito com foco em aprendizado.
